
# Healthcare Operations & Billing Analytics Dashboard

## Project Overview

This project presents a Healthcare Analytics dashboard built using Power BI. The dashboard analyzes hospital operations, patient demographics, admission patterns, insurance providers, and billing trends to generate actionable insights for healthcare decision-making.

## Objectives

* Analyze patient admissions and healthcare service utilization
* Identify trends in hospital billing and treatment costs
* Examine the relationship between patient age and treatment cost
* Evaluate hospital stay duration across insurance providers
* Visualize clinical outcomes through test results and health conditions

## Tools Used

* Power BI
* Power Query
* Data Visualization

## Key Metrics Included

* Total Patients
* Average Length of Hospital Stay
* Total Billing Amount
* Average Billing Amount by Admission Type
* Patient Distribution by Gender
* Test Results Distribution
* Cost vs Age Patterns by Medical Condition
* Billing Trend by Year

## Key Insights

* Emergency admissions tend to show higher average billing amounts compared to elective and urgent admissions.
* Billing amounts show noticeable variation across different years.
* Insurance providers influence the average hospital stay duration.
* Patient age demonstrates varying treatment cost patterns depending on the medical condition.
